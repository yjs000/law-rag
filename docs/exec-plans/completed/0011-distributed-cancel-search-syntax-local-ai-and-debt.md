# 분산 취소·검색 문법·로컬 AI·부채 감사

기준 시점: 2026-07-18  
상태: 완료

## 목적과 사용자 결과

- sticky routing 없이 어느 Vercel 인스턴스가 취소 요청을 받아도 분산 취소 의도가 유실되지 않는 계획을 확정한다.
- 한글 수사, 복수 항, 조문 범위를 검색 전용 모드에서 결정적으로 처리한다.
- Windows PC 로컬 모델을 Vercel 서비스와 연결할 수 있는 안전한 대안과 사용자 선행 작업을 공식 자료로 확인한다.
- 임베딩과 Qwen 추가 구현의 현재 상태를 설명하고 프로젝트 기술·로직 부채를 재감사한다.

## 사용자 병목 우선순위

1. **배포 경로 결정(P0)**: 사용자 PC가 공개 인바운드를 받지 않는 현재 아키텍처와 Vercel→로컬 모델 호출 요구가 충돌한다. 안전한 사설 연결/중계 또는 API 배치 변경을 사용자가 선택해야 한다.
2. **공유 취소 저장소(P1)**: 분산 취소에는 Supabase PostgreSQL 등의 영속 신호 저장소와 migration 적용 권한이 필요하다.
3. **로컬 하드웨어(P1)**: NVIDIA GPU 모델·VRAM, Windows/WSL 사용 가능 여부에 따라 실행 가능한 모델과 runtime이 달라진다.
4. **임베딩 공급자(P1)**: 기존 OpenAI 임베딩 유지 여부에 따라 비용·키·재색인 작업이 달라진다.

## 범위와 비범위

- 범위: 설계와 상태 계약, parser/repository 테스트, 공식 NVIDIA 조사, 설정 체크리스트, 부채 문서, 외부 자격정보 없이 가능한 코드·mock 테스트.
- 비범위: 사용자 Supabase migration 실행, DNS/TLS/터널 계정 생성, NVIDIA 드라이버 설치, 실제 Qwen 다운로드와 운영 배포.

## Agent별 단계

- [x] 분산취소 Agent: 공유 취소 신호·404 상태 계약·분산 테스트 계획과 문서.
- [x] 검색문법 Agent: 한글 수사·복수 항·범위 구현과 회귀 테스트.
- [x] 로컬모델 Agent: NVIDIA 공식 자료 조사와 사용자 실행 체크리스트.
- [x] Root Agent: 임베딩 설명·Qwen 구현 상태·전체 부채 감사·통합 검증.

## 완료 조건

- [x] cancel endpoint가 404를 반환하는 조건과 정상적인 비동기 취소 상태를 구분하는 계획이 있다.
- [x] 세 가지 신규 조문 표현의 정상·실패·경계 테스트가 통과한다.
- [x] 로컬 PC 대안의 보안·비용·하드웨어·사용자 작업이 공식 근거와 함께 기록된다.
- [x] 기술·로직 부채가 우선순위·영향·종료 조건과 함께 tracker에 반영된다.
- [x] 변경된 코드의 관련 테스트·린트·타입 검사가 통과한다.

## 결과

- 분산 취소는 Supabase 영속 상태+polling watcher로 설계하고 memory coordinator의 두 인스턴스 mock을 검증했다. 운영 migration은 별도 실행 계획 0012에 남겼다.
- 한글 수사·복수 항·최대 20개 범위 검색을 구현했다.
- 현재 GTX 1650/Windows 10은 NVIDIA NIM on WSL2 공식 지원선 밖임을 확인했고 outbound queue를 우선 대안으로 정리했다.
- 운영 설정 누락 거부, 대화 컨텍스트 중복 보존 방지, 직접 API 컨텍스트 상한과 prompt 신뢰 경계를 개선했다.
- 검증: Core 4, API 192 테스트, Ruff, 문서 79개 검사가 통과했다.

## 검증과 롤백

- 검색 변경은 API 전체 pytest와 Ruff를 실행한다.
- Web/API 계약 변경이 있으면 Vitest·TypeScript·ESLint·build를 실행한다.
- DB migration과 외부 서비스 설정은 mock/설계까지만 수행하고 사용자 승인·자격정보 없이 적용하지 않는다.

## 결정 로그

- 2026-07-18: sticky routing은 사용하지 않는다.
- 2026-07-18: 분산 취소의 404는 단순 “다른 인스턴스”를 뜻하지 않게 하고, 공유 상태에서 owner 불일치/존재하지 않는 ID만 404로 제한하는 방향을 검토한다.
- 2026-07-18: 법률 검색 범위 표현은 임의 확장하지 않고 파싱된 각 정확 경로를 기준일·법령 범위 안에서 조회한다.

## 미결정과 차단 요소

- 사용자 GPU 모델/VRAM과 인터넷 공개 정책.
- Supabase migration을 실제 운영 DB에 적용할 시점.
- 로컬 생성 endpoint를 사설 중계할 방식과 인증서/도메인 소유.
