# 데이터베이스 스키마

> 상태: 아직 생성되지 않음  
> 최종 확인: 2026-07-13  
> 생성 명령: 기술 스택 확정 후 이 머리글에 기록

이 파일은 마이그레이션에서 자동 생성되는 스키마 문서의 자리표시자다. 수동 편집을 권위 있는 스키마 변경으로 사용하지 않는다.

## 예상 핵심 엔터티(비확정)

- sources: 데이터 출처와 이용/갱신 정책
- ingestion_runs: 수집 실행과 상태
- legal_documents: 법률 문서의 안정적 정체성
- document_versions: 기준일과 원본 해시를 가진 버전
- chunks: 원문 위치를 보존한 검색 단위
- index_versions: 색인 구성과 승격 상태
- queries/answers/citations: 질의 실행, 답변, 원문 범위 연결
- evaluation_runs: 평가 구성과 결과
- audit_events: 민감한 관리 작업의 감사 기록

실제 테이블·컬럼·관계는 ADR과 마이그레이션이 도입된 후 생성한다.
