# 신규 사용자 온보딩

상태: 가정 기반 초안  
작성일: 2026-07-13

## 사용자와 문제

초기 사용자는 법률 자료를 찾고 근거를 빠르게 확인하려는 한국어 사용자로 가정한다. 사용자는 서비스가 법률 자문인지, 어떤 자료를 대상으로 하는지, 답변을 얼마나 신뢰해야 하는지 알기 어렵다.

## 사용자 결과

사용자는 첫 질문 전에 서비스의 범위와 한계를 이해하고, 첫 답변에서 인용 원문을 열어 확인할 수 있다.

## 흐름

1. 지원 자료, 기준일, 법률 자문 대체 불가 고지를 짧게 본다.
2. 예시 질문 또는 직접 질문으로 시작한다.
3. 사건 식별 가능 정보 입력을 피하라는 안내를 받는다.
4. 답변의 근거 표시 방법을 한 번 안내받는다.
5. 인용을 열어 원문 위치와 출처를 확인한다.

## 수용 기준

- 첫 화면에서 지원 관할·자료 범위·기준일을 찾을 수 있다.
- 고지는 질문을 방해하지 않지만 제출 전에 인지 가능하다.
- 예시 질문에는 실제 개인 사건정보가 포함되지 않는다.
- 첫 답변의 모든 인용은 유효한 원문 위치로 이동한다.
- 사용자는 온보딩 안내를 다시 열 수 있다.

## 미결정

- 로그인 필요 여부
- 일반인/전문가별 별도 온보딩
- 동의가 필요한 개인정보 처리 범위
