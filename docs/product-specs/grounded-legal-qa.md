# 분산에너지 사업 법령 조사 워크벤치

## 사용자와 문제

초기 사용자는 분산에너지 사업을 검토하는 일반 사용자다. 사용자는 법령 이름보다 “지금 이 사업 단계에서 무엇을 허가·신고·점검해야 하는가”를 알고 싶다. 서비스는 법률 자문이나 승패 예측 대신 조사 출발점과 원문 경로를 제공한다.

## 핵심 여정

1. 사업 단계와 법령 기준일을 선택하고 자연어로 질문한다.
2. 서비스가 허용 코퍼스에서 기준일 유효 조문을 검색한다.
3. 답변의 의무·예외·적용 조건 옆에서 인용 ID를 확인한다.
4. 원문 패널에서 정확한 법령 버전과 조문을 대조한다.
5. AI가 없거나 근거가 부족하면 생성 결론 대신 검색 결과와 한계를 본다.
6. 목업 Google 계정으로 로그인한 사용자는 자신의 질문 이력을 다시 열 수 있고, 익명 사용자의 질문은 저장되지 않는다.
7. 사업 단계 체크리스트를 Markdown을 기본으로, CSV 또는 PDF를 선택해 내보낸다.
8. 익명 사용자가 첫 질문의 답변을 받은 뒤 `질문 기록을 남기려면 로그인하세요` 팝업을 보고 Google 목업 로그인으로 이동할 수 있다.

## MVP 허용 목록

전기사업법과 시행령·시행규칙, 분산에너지 활성화 특별법과 시행령, 신에너지 및 재생에너지 개발ㆍ이용ㆍ보급 촉진법, 전기안전관리법, NFPC 607, NFTC 607의 정확 명칭 9개다.

## 완료 기준

- 질문·단계·기준일 입력과 답변/원문 패널
- 주장별 존재하는 인용 ID와 정확 원문 반환
- 코퍼스 상태와 검색 전용 모드 표시
- 기준일 유효 버전 검색
- AI 쿼터 오류의 안전한 폴백
- 법률 자문 대체 불가 고지
- 목업 로그인과 로그인 사용자 전용 질문 이력
- 첫 익명 질문 응답 후 세션당 한 번만 표시되는 비차단 로그인 유도 팝업
- 질문 이력의 1년 만료와 계정 삭제 시 사용자 관련 데이터 전체 삭제
- 동일한 체크리스트 데이터에서 Markdown·CSV·PDF 생성
- 표지·브랜딩 없는 단순 PDF 출력본

## 비목표

판례·블로그·다른 법률 사이트 검색, 개인 사건 판단, 법적 승패 예측, PDF 기본 청킹, 모델 기억으로 누락 근거 보완은 지원하지 않는다.
