# 근거 기반 법률 질의

상태: 가정 기반 초안  
작성일: 2026-07-13

## 사용자 이야기

법률 자료를 조사하는 사용자로서, 질문과 관련된 법령·판례를 근거 위치와 함께 보고 싶다. 그래야 결론을 독립적으로 확인하고 다음 조사를 이어갈 수 있다.

## 범위

포함:

- 한국어 자연어 질문
- 관할·기준일·문서 유형 필터
- 근거 기반 요약과 주장별 인용
- 원문 위치 보기
- 근거 부족·충돌·비지원 범위 표현
- 답변 품질 피드백

제외(MVP):

- 사건 승패 예측
- 소송 서면의 자동 제출
- 변호사-의뢰인 관계 형성
- 공개 이용 조건이 확인되지 않은 자료 전문 제공

## 기능 요구사항

1. 질문 제출 전 길이, 형식, 정책을 검증한다.
2. 답변은 적용 관할, 기준일, 검색한 자료 범위를 표시한다.
3. 핵심 주장에는 하나 이상의 검증된 인용이 연결된다.
4. 인용은 출처, 문서명, 날짜, 조문/사건번호/문단 위치를 표시한다.
5. 근거가 부족하면 필요한 추가 정보나 지원 범위를 알린다.
6. 같은 질문과 같은 버전 조합은 재현 가능한 검색 결과를 만든다.
7. 사용자는 답변과 개별 인용에 오류 피드백을 남길 수 있다.

## 실패 상태

- 질문이 너무 넓음: 범위를 좁힐 선택지 제시
- 개인 식별정보 감지: 최소화 안내와 편집 기회 제공
- 관련 자료 없음: 검색 범위와 대체 검색어 표시
- 출처 충돌: 양쪽 근거와 충돌 사유 표시
- 생성 실패: 검색된 근거 목록을 제공하거나 안전하게 재시도

## 수용 기준

- 존재하지 않는 인용 ID나 원문 범위는 사용자에게 노출되지 않는다.
- 근거 없는 핵심 주장이 있으면 답변 검증이 실패한다.
- 기준일 필터가 문서의 효력 기간에 적용된다.
- 운영 로그만으로 사용자의 전체 민감 질문을 복원할 수 없어야 한다.
- 대표 사용자 여정이 접근성 있는 키보드 흐름으로 완료된다.
