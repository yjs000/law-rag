## 갱신 규칙

- 중요한 릴리스 전과 월 1회 갱신한다.
- 등급에는 테스트, 대시보드, 평가 실행 등 검증 가능한 근거를 연결한다.
- 점수를 높이는 것보다 격차와 다음 종료 조건을 정확히 쓰는 것을 우선한다.

## 현재 평가 해석

5개 답변 품질 사례는 외부 API, 실제 색인, Terra 모델, 실제 사용자 데이터를 사용하지 않는 회귀용 목업이다. 별도의 Production 검색 시드 8개는 실제 색인에서 기대 문서 포함 또는 근거 부족 계약을 통과했지만, 1회 소표본이고 AI 생성은 실행하지 않았다. 두 결과 모두 법률적 정답률을 입증하지 않으며 Recall@k·MRR/nDCG, 블라인드 평가와 법률 전문가 검토는 아직 미실행이다.
