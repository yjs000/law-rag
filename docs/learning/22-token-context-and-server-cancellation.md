# 토큰 컨텍스트와 서버 작업 취소

기준 시점: 2026-07-18

메시지 수는 모델 입력 비용을 나타내지 않는다. 짧은 100개 메시지보다 긴 법령 원문 한 턴이 더 비쌀 수 있으므로, 현재 질문과 완료된 최근 질문·답변의 보수적 토큰 추정 합계를 사용한다. 기본 입력 예산은 24,576이며 Qwen3-4B native 32K 안에서 출력 4,096과 system·근거·schema 여유 4,096을 별도로 확보한다.

이전 답변은 대화 맥락일 뿐 법률 근거가 아니다. 생성 prompt에 이를 명시하고 현재 검색 결과의 C번호로 실질 주장을 다시 검증한다. 중단·오류 응답은 다음 컨텍스트에 넣지 않는다.

브라우저의 `AbortController`는 연결만 끊을 수 있다. 서버는 client request UUID와 소유자 키로 현재 asyncio task를 등록하고, 별도 취소 endpoint가 `Task.cancel()`을 호출해야 검색·모델 await에 취소가 전파된다. 완료·미등록·다른 소유자 요청은 모두 404로 처리하고 `finally`에서 레지스트리를 정리한다.

현재 레지스트리는 프로세스 로컬이다. 다중 Vercel 인스턴스에서 취소 요청이 다른 인스턴스로 가면 찾을 수 없으므로 운영에서는 sticky routing 또는 Redis/PostgreSQL 취소 신호와 worker 협력 취소가 필요하다. blocking SDK/native 호출은 await 제어가 돌아올 때까지 취소가 늦어질 수 있다.
