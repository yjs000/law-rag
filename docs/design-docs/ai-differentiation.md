# AI 차별화와 안전 설계

일반 챗봇의 장점은 자연스러운 요약이고 약점은 법령 시점과 근거 추적이다. 이 제품은 AI를 결론 저장소가 아니라 검색된 규제 경로를 읽기 쉬운 조사 결과로 변환하는 계층으로 사용한다.

차별점은 문장별 조문 인용, 기준일 버전, 법률 계층, 사업 단계 체크리스트, 변경 영향이다. 모델은 제공된 근거 ID만 출력할 수 있고 검증 실패 시 생성 결과 전체를 버린다. quota 소진 때도 원문 검색은 유지된다.

AI 효과가 큰 작업은 질문 구조화, 여러 근거의 조건/예외 합성, 의무 체크리스트 초안, 개정 영향 설명이다. 법령 존재 확인, 버전 선택, 인용 위치 확인은 결정론적 코드가 맡는다.
