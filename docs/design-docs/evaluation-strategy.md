# 평가 전략

상태: 승인 전 초안
작성일: 2026-07-13
최종 갱신: 2026-07-15

## 원칙

평가는 검색, 답변 근거성, 법률 정확성, 안전성, 사용자 효용을 분리한다. 모델 심사만으로 출시를 결정하지 않으며 전문가 표본 검토와 결정적 검사를 포함한다.

## 평가셋 구성

- 문서 유형과 난이도별 대표 질문
- 정확한 조문/사건번호 탐색 질문
- 여러 문서를 결합해야 하는 질문
- 기준일에 따라 답이 달라지는 질문
- 정답 근거가 코퍼스에 없는 질문
- 상충하는 판례·해석이 있는 질문
- 개인정보, 범죄 조력, 프롬프트 주입 등 안전 사례

평가셋은 `개발`, `회귀`, `블라인드`로 분리하고 실제 사용자 데이터는 비식별화와 사용 근거가 확보된 경우에만 편입한다.

### 결정적 답변 계약 시드

`apps/api/tests/fixtures/answer_quality_cases.json`은 실제 사용자 입력이나 외부 API 호출 없이 실행하는 개발·회귀용 시드다. 국가법령정보 공동활용 Open API 허용 목록에 속하는 문서의 기존 fixture 원문 또는 명시적 목업 `SearchHit`만 사용한다.

현재 시드는 다음 5개 질문 유형을 포함한다.

| 유형 | 평가하는 판단 | 기대 근거 예시 |
|---|---|---|
| 허가 | 허가 주체와 규범어를 원문 범위에서 제시 | 전기사업법 제7조 제1항 |
| 신고/변경 | 중요사항 변경허가 근거를 신고만으로 충분하다고 확대하지 않음 | 전기사업법 제7조 제1항 |
| 전기저장시설 안전 | 성능기준의 목적·적용범위를 안전 보장으로 과장하지 않음 | NFPC 607 제1조·제2조 |
| 직접 조문 경로 | 법령명과 정규화된 조문 경로를 그대로 왕복 | 전기사업법 제7조/항① |
| 범위 밖/근거 없음 | 판례·개인 사건 승패를 단정하지 않고 근거 없음 상태를 반환 | 인용 없음 |

각 사례는 질문, 기준일, 사업 단계, 목업 원문, 기대 문서명·조문 경로, 참고 답변, 필수 용어, 금지할 과잉 결론을 함께 고정한다. 참고 답변은 생성 모델의 유일한 정답 문구가 아니라 최소 의미 계약이다.

## 지표

| 계층 | 핵심 지표 |
|---|---|
| 수집 | 파싱 성공률, 메타데이터 완전성, 최신성 지연 |
| 검색 | Recall@k, MRR/nDCG, 근거 없음 탐지율 |
| 답변 | citation precision/coverage, 근거 충실도, 정확성 |
| 안전 | 개인정보 누출, 비지원 자문 거절의 정밀도/재현율 |
| 운영 | p50/p95 지연, 오류율, 질의당 비용 |

결정적 시드 테스트는 다음 지표를 모두 통과해야 한다.

- 목업 근거 문서의 허용 목록 준수율 100%
- 기대 문서 집합과 조문 경로 집합의 정확 일치 100%
- 응답 인용문과 제공된 목업 원문의 정확 일치 100%
- 참고 답변과 검색 전용 응답의 필수 용어 포함률 100%
- 금지된 과잉 결론 발생 0건
- 결과 상태와 근거 없음 사유 코드 정확 일치 100%

이 수치는 작은 결정적 목업 계약의 회귀 통과율이다. 실제 색인의 Recall@k, Terra 답변의 법률 정확성, 다양한 표현에 대한 일반화 성능을 뜻하지 않는다.

## 릴리스 게이트 초안

- 인용 ID와 원문 범위 검증: 100%
- 기존 블라인드 셋의 중대 오류: 증가 금지
- 검색 Recall@k와 인용 커버리지: 기준 버전 대비 허용 하락폭 이내
- 안전 핵심 사례: 모두 통과
- 목표 수치는 초기 베이스라인 측정 후 확정

## 결과 기록

모든 평가 실행은 코드 SHA, 데이터셋 버전, 색인 버전, 모델/프롬프트 버전, 설정, 비용과 지연을 남긴다. 실패 사례는 원문 질문을 무조건 저장하지 말고 개인정보 정책을 먼저 적용한다.

현재 5개 시드는 외부 코퍼스와 모델을 호출하지 않으므로 비용·지연·색인 버전·모델 버전 측정 대상이 아니다. 실제 코퍼스 회귀셋과 Terra 출력 평가를 실행할 때 위 실행 메타데이터를 별도로 기록한다. 법률 전문가 표본 검토와 블라인드 평가는 아직 수행하지 않았다.

## 결정 기록

- 2026-07-15: 답변 품질 작업의 첫 결정적 하한선으로 5개 목업 질문-근거-답변 계약을 추가했다. 문서·경로·원문·필수 용어·과잉 결론을 코드로 검사하되, 이를 실제 검색 품질이나 법률 정확성의 증거로 확대 해석하지 않는다.
