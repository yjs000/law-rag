# 시간 효력 모델

질문 `as_of_date`는 공포일이 아니라 시행 효력에 적용한다. 버전은 `effective_from <= as_of_date < effective_to`일 때 유효하다. 종료일이 없으면 현재/시행예정 상태를 날짜로 판정한다.

`eflaw`는 시행일 기준 현행 검색에 사용한다. `target=law`는 공포일 기준 진단과 과거 MST 본문 확인에만 쓴다. 시행일, 공포일, 수집일을 서로 대체하지 않는다.

경계일, 과거, 현재, 시행예정 fixture를 별도 테스트해야 한다. 현재 MVP 스키마와 검색 필터는 이 계약을 구현했으나 과거 버전 전체 수집은 아직 활성화하지 않았다.
